

#Vue3 + ag-grid

Este proyecto esta hecho con vue3 + ag-grid utilizando composition API
ag-grid es de pago, pero ofrece muchas funcionalidades de manera gratuita.

## Librerias:
VUE 3 
https://vuejs.org/

AG GRID
https://www.ag-grid.com/

API PUBLICA PARA PRUEBAS

https://api.publicapis.org/entries

# Al clonar el proyecto:
## Ejecutar
```
npm install
```
### Levantar el proyecto
```
npm run serve
```
